# AI Guardian - Frontend

This directory contains the web-based dashboard for AI Guardian.

## Files

- `index.html` - Main dashboard with real-time metrics and alerts

## Features

- Live video feed from patient camera
- Real-time health metrics (heart rate, breathing, stress, tremor)
- Safety metrics (fall risk, self-harm risk, aggression, dangerous objects)
- Disease risk assessment (Parkinson's, sleep apnea, diabetes, BP)
- Active alerts with severity levels
- Overall risk level assessment

## Deployment

Open `index.html` in a web browser to access the dashboard. Ensure the backend API is running on `http://localhost:5000`.
